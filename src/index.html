Prizm Energy ERP — Outlook add-in

One menu in every email, hooked into your Perfex ERP. Create tasks, opportunities, leads, tickets, link emails to records — without leaving Outlook.

Heads-up: Microsoft disabled "Add from URL" for custom add-ins in early 2026 — you now have to download the manifest and use "Add from File". The button below does step 1 for you.
Step 1 — Download the manifest
A single XML file. Save it anywhere; you'll point Outlook at it once.
Download

Step 2 — Add to Outlook

  1. Outlook on the web: Click Get add-ins (the puzzle-piece icon) → My add-ins → Add a custom add-in → Add from File... → pick the manifest you just downloaded.
  2. Outlook desktop (Windows/Mac): File → Get Add-ins → My add-ins → Add a custom add-in → Add from File...
  3. Confirm any "this add-in could access info" prompt → done.

Step 3 — Get your API key

  1. Sign in to Prizm ERP → sidebar Outlook → Add-in API keys (production).
  2. Click Generate new key → copy the token shown (it's revealed once).
  3. In Outlook, open any email → Prizm ERP → Open Prizm ERP → Settings → paste the key → Test connection.

That's it

From here on, any email shows a Prizm ERP action in its action menu with sub-items for every supported record type. New ERP actions appear automatically when we ship them — no re-install needed.

Trouble?

  • If Test connection returns a red banner: confirm the API base URL in Settings is https://ms.prizm-energy.com/MS/outlookapi/bridge (or https://dev.prizm-energy.com/outlookapi/bridge for the dev environment).
  • If the menu doesn't appear after sideload: reopen the email (or restart Outlook), then check My add-ins shows Prizm Energy ERP as installed.
